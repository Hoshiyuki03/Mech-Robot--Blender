A project by an aspiring game designer exploring 3D modeling and animation.
This project features a low-poly mech robot model created in Blender 4.5.1 LTS. It was built as part of my journey to learn 3D design, animation, and environment setup for game development.
The mech was designed with a focus on clean topology, modular parts, and a stylized look suitable for integration into game engines like Unity or Unreal Engine.
